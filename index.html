API Documentation

Here is the documentation for ChessPuzzlesAPI along with code examples:

Example 1: Getting all the puzzles

                
                    import requests

                    # Define the API endpoint URL
                    api_url = "https://untitled-master.github.io/chesspuzzlesapi/data.json"
                    
                    try:
                        # Send a GET request to the API
                        response = requests.get(api_url)
                    
                        # Check if the request was successful (status code 200)
                        if response.status_code == 200:
                            # Parse the JSON response
                            data = response.json()
                    
                            # Now 'data' contains the JSON data from the API
                            # You can work with the data as needed
                            for puzzle in data:
                                print(f"Puzzle ID: {puzzle['PuzzleId']}")
                                print(f"FEN Position: {puzzle['FEN']}")
                                print(f"Moves: {puzzle['Moves']}")
                                print(f"Rating: {puzzle['Rating']}")
                                # ... and so on for other fields
                    
                        else:
                            print(f"Failed to retrieve data. Status code: {response.status_code}")
                    
                    except requests.exceptions.RequestException as e:
                        print(f"Request error: {e}")

Example 1: Getting a random puzzle

                
                    import requests
                    import random
                    
                    # Define the API endpoint URL
                    api_url = "https://untitled-master.github.io/chesspuzzlesapi/data.json"
                    
                    try:
                        # Send a GET request to the API
                        response = requests.get(api_url)
                    
                        # Check if the request was successful (status code 200)
                        if response.status_code == 200:
                            # Parse the JSON response
                            data = response.json()
                    
                            # Choose a random puzzle from the list
                            random_puzzle = random.choice(data)
                    
                            # Now 'random_puzzle' contains the JSON data of a random puzzle
                            print(f"Random Puzzle ID: {random_puzzle['PuzzleId']}")
                            print(f"Random FEN Position: {random_puzzle['FEN']}")
                            print(f"Random Moves: {random_puzzle['Moves']}")
                            print(f"Random Rating: {random_puzzle['Rating']}")
                            # ... and so on for other fields
                    
                        else:
                            print(f"Failed to retrieve data. Status code: {response.status_code}")
                    
                    except requests.exceptions.RequestException as e:
                        print(f"Request error: {e}")
